#basic sets of terminal commands.

Navigate to padho-ai-enginer folder : cd padho-ai-engineer
Navigate to week folder : cd week_number
Create that day folder : uv init day_no (uv init day2)
Copy env file just simple ctrl c and ctrl v
Now navigate to day_no folder before creating virtual environment : cd day_no
To create virtual environment inside its folder: uv venv --python 3.11
Activate the virtual environment :  .\.venv\Scripts\activate.ps1
Create .py file : code file_name.py (code sys_temp.py)
To run the file : python .\file_name.py (python .\sys_temp.py)
Always import dotenv file : uv add python-dotenv
Also groq :  uv add groq
